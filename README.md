Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @jasoeight/number-formatter`

## Usage

    var numFormatter = require('@jasoeight/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm run test`
